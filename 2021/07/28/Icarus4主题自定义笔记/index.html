Icarus4主题自定义笔记

icarus主题简洁大方，提供的配置项足够使用。不过我在此基础上对它进行小小的一番自定义，使其更加符合我的需求。

修改网站主题色

用你喜欢的编辑器打开 node_modules\bulma-stylus\stylus\utilities\initial-variables.styl ，便可看到主题所使用的的一些颜色变量。按照stylus的语法，模仿已有配置添加你想要的颜色变量。

打开 node_modules\hexo-theme-icarus\include\style\base.styl ，修改其中的 $primary 变量，使其指向你刚才添加的颜色变量。

打开 node_modules\bulma-stylus\stylus\utilities\derived-variables.styl ，修改其中的 $link 变量，使其指向你刚才添加的颜色变量。

保存这些文件，运行 hexo clean && hexo g && hexo s 在本地查看修改效果。
